# MAZESTA TOOLS

Advanced Termux installer and tool manager.

## Install

```bash
git clone https://github.com/Dr-Mazesta/MAZESTA-TOOLS
cd MAZESTA-TOOLS
chmod +x install.sh run-tools.sh wifi-tools.sh
./install.sh
```

## Run

```bash
./run-tools.sh
```

- Termux tools  
- Network tools  
- Security tools
